Guide interactif QGIS

Bienvenue dans le lanceur de guides

Cette page sert d'exemple pour valider le fonctionnement du plugin. Tu peux la remplacer par tes vrais parcours utilisateurs métier.

Principe

Le plugin ouvre ce guide dans une fenêtre indépendante. L'utilisateur peut donc continuer à manipuler QGIS sans perdre la main sur la carte, les couches ou les formulaires.

Organisation

Place tes fichiers HTML dans le dossier web/guides. Les feuilles de style, scripts et images peuvent être rangés dans web ou dans un sous-dossier dédié.

Étapes suivantes

  1. Créer un guide par procédure métier.
  2. Ajouter des boutons précédent / suivant en JavaScript.
  3. Brancher plus tard le HTML avec PyQGIS si tu veux piloter QGIS depuis le guide.
