Guide pratique QGIS

Mettre QGIS en français

Ce guide simple permet de tester la mise en page tout en couvrant une action fréquente: passer QGIS en francais.

Sommaire

  • Etape 1 - Ouvrir les options
  • Etape 2 - Activer le remplacement
  • Etape 3 - Choisir la langue
  • Etape 4 - Valider et redemarrer
  • Verification rapide
  • Depannage

Etape 1

Ouvrir les options de QGIS

Ce guide part du principe que QGIS en anglais.
Aller dans le menu Settings > Options

Etape 2

Activer le remplacement des paramètres régionaux

Vous devriez être dans l'onglet Generale par défaut si ce n'est pas le cas il faut vous rendre à l'intérieur.
Cocher la case Override System Locale.

Etape 3

Choisir la langue

Cherchez la section Language. Séléctionner votre langue dans le menu déroulant User interface translation

Etape 4

Valider et redemarrer QGIS

Cliquez sur le bouton OK pour sauvegarder les changements. Redémarer Qgis pour que le changement de langue soit effectif.

Verification rapide

  • Les menus de QGIS sont dans la langue choisie.
  • Les boites de dialogue principales sont traduites.
  • Les extensions tierces peuvent rester dans leur langue d'origine.

Depannage

Si la langue ne change pas, retourne dans Parametres > Options, verifie la case de locale et refais un redemarrage complet de QGIS.
